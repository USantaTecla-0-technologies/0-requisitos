# Textos

[Volver](../README.md)

## Ecos

### Eco simple

* Mostrar por pantalla la propia entrada del usuario
  * &lt;mensaje&gt;

### Eco triple

* Mostrar por pantalla la entrada del usuario en diferentes ventanas:
  * &lt;mensaje&gt;!!!
  * &lt;mensaje&gt;!!
  * &lt;mensaje&gt;!

## Saludos

### Hola Mundo

* Mostrar:
  * "Hola Mundo"

### Saludo Mundo

* Mostrar:
  * "Hola, Mundo!!!"
  * "Disculpa, no me despedí!!!"
  * "Hasta luego!!!"

### Hola personalizado

* Solicitar el nombre del usuario
* Mostrar:
  * "Hola, &lt;nombre&gt;!!!"

### Saludo personalizado

* Solicitar el nombre del usuario
* Mostrar:
  * "Hola, &lt;nombre&gt;!!!"
  * "Adios, &lt;nombre&gt;!!!"

## Morse

### Morse

* Dado un texto, traducirlo al código Morse según la [wiki](https://es.wikipedia.org/wiki/C%C3%B3digo_morse#/media/Archivo:International_Morse_code.png)
